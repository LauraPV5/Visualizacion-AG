Un gráfico de barras horizontal representa visualmente datos recolectados en barras horizontales. El uso de éstos gráficos radica en un potencial de comparación de las categorías o las cualidades de una variable.

Como se ha dicho, hace que en un solo vistazo puedan hacerse comparaciones, generalizaciones sobre los datos incluso deducir tendencias. Se generan fácilmente gracias al salto directo de una hoja de cálculo e los ejes X e Y. Además, la comparación entre las distintas categorías se facilita cuando las barras se ordenan según la frecuencia y la búsqueda de una categoría es facilitada por la ordenación alfabética de las mismas incluso el hecho de poner el valor numérico de cada barra es punto favorable para la comprensión y visualización.

Cabe añadir que la lectura del eje X y del eje Y se dificulta cuando se representan muchas categorías o cualidades, además no son adecuados para comparar muchas categorías. Dificulta la comparación si se considera una evolución en el tiempo ya que supone crear diferentes gráficos por tramo de tiempo y categoría. Punto a tener en cuenta es que si se quiere comparar más de una variable, se necesita crear un gráfico para cada una.

En resumen, éstos gráficos tienen una capacidad comparativa visual.

Ejercicio de ampliación

Gráficos combinados. Múltiples variables.

Los gráficos combinados solo admiten una dimensión lo cual implica que no pueden usarse cuando se necesitan incluir dos o más dimensiones. Además solo pueden tener dos ejes de medida y en caso de agregar tres o más con bastante diferencia en el rango de valores complica la situación a la hora de mostrar las medidas con una distribución de valores buena. Por tanto con éstos gráficos se dificulta el entendimiento de los datos.



Datawrapper

Datawrapper es una herramienta de visualización en línea a través de la cual todo el mundo puede crear gráficos, mapas y tablas. Se puede incluir la visualización final en un sitio web propio o incluso exportarla como un PDF entre otras. Posee una interfaz intuitiva y todo es en el navegador personal, es decir, no es necesario descargarse una aplicación. Es muy bueno a la hora de visualizar datos en los gráficos que tienen pero si se quiere una herramienta para crear visualizaciones más personalizadas, datawrapper no es la indicada. Otro contra es que se tiene que limpiar los datos antes de subirlos a Datawrapper ya que la aplicación es para visualizar datos pero no para analizarlos. Se ha realizado un diagrama de dispersión donde resaltamos varios paises como España, China y Estados Unidos, estos últimos con mayor número de emisiones.
